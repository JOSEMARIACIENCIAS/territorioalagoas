# Como enviar imagens manualmente para o GitHub

1. Compacte todas as imagens em um arquivo chamado `imagens.zip`.
2. No GitHub, clique em "Adicionar arquivo" > "Enviar arquivos".
3. Faça upload do `imagens.zip` e clique em "Commit changes".

**Atenção:** O GitHub não descompacta automaticamente. Para que as imagens fiquem acessíveis no site, é necessário descompactar e enviar cada imagem individualmente pelo site ou usar o git localmente.

## Recomendações
- Prefira nomes de arquivos sem acentos, espaços ou caracteres especiais.
- Se possível, use o git no computador para enviar pastas inteiras de forma mais fácil e rápida.

Se precisar de ajuda para usar o git, peça instruções aqui.
