# Como enviar arquivos de áudio manualmente para o GitHub

1. Compacte o(s) arquivo(s) de áudio (exemplo: audio/audiominha-alagoas.mp3) em um arquivo chamado `audio.zip`.
2. No GitHub, clique em "Adicionar arquivo" > "Enviar arquivos".
3. Faça upload do `audio.zip` e clique em "Commit changes".

**Atenção:** O GitHub não descompacta automaticamente. Para que o áudio fique acessível no site, é necessário descompactar e enviar o arquivo mp3 individualmente pelo site ou usar o git localmente.

## Recomendações
- Prefira nomes de arquivos sem acentos, espaços ou caracteres especiais.
- Se possível, use o git no computador para enviar pastas inteiras de forma mais fácil e rápida.

Se precisar de ajuda para usar o git, peça instruções aqui.
